# 项目简介

## 核心功能
- 基于YOLOv8的腐蚀实例分割检测系统
- 支持多种输入源检测：单张图片、图片文件夹、视频文件、本地摄像头
- 提供自定义功能：可调整置信度、IOU阈值，选择是否显示标签和原图
- 支持选择不同检测模型
- 实时显示目标位置、目标总数、置信度等信息
- 支持批量检测并直接查看所有检测结果
- 支持检测结果保存

## 适用场景
- 腐蚀检测与识别
- 工业设备维护
- 材料质量评估
- 自动化检测系统
- 实时监控应用

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于python3.9、opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 提供1600张腐蚀类数据集，包含1个分类
- 包含完整源码、UI界面源文件、图标、数据集图片和项目运行教程
- 已训练好模型，配置成功环境后可直接使用
